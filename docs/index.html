sort

Sorts the array using a given compare function. If not specified, the elements will be sorted according to their unicode value.

sort
Parameters
array (Array<any>) The target array.
compare (function (current: any, next: any): number) The function which compares the current element to the next . If it returns a negative value current before next , if it returns a positive value current comes after next , otherwise they're left unchanged.
Returns
Array<any>: A new sorted array.
Example
import { reduce } from "@srcode/aide";

// default compare function
sort([1, 5, 2]); // [1, 2, 5]

// custom compare function
const largestFirst = (current, next) => next - current;
sort([1, 5, 2]); // [5, 2, 1]

mapKeys

Calls a functions for each key of the given object, returning a new key.

mapKeys
Parameters
object (Object) The target object.
keyFn (function (key: any, value: any): any) The function to call for each key, returning a new key.
recursive (boolean) If true, will recursively apply the functions to nested objects as well.
Returns
Object: A new object with the transformations applied.
Example
import { mapKeys } from "@srcode/aide";

mapKeys({ first: 51, second: 24 }, key => key.toUpperCase()); // { FIRST: 51, SECOND: 24 }

get

Returns the value at the given path of the given object.

get
Parameters
object (Object) The target object.
path (string) The path to be accessed.
defaultValue (any) The value to be returned if the value cannot be found.
Returns
any: The value at the given path.
Example
import { get } from "@srcode/aide";

const obj = { employees: { "51213": { children: [{ name: "Joe" }] }}};
get(obj, "employees["51213"].children[0].name"); // "Joe"

set

Sets the value at the given path of the given object. If path does not exists, it's expanded (object for string keys, array for numeric keys) until the value is set.

set
Parameters
object (Object) The target object.
path (string) The path where the value will be set.
value (any) The value to set.
Returns
Object: A new object with the path set to the given value.
Example
import { set } from "@srcode/aide";

const unchanged = { employees: { "51213": { children: [{ name: "Joe" }] }}};
const changed = set(unchanged, "employees["51213"].children[0].name", "Bob"); // { employees: { "51213": { children: [{ name: "Bob" }] }}}
